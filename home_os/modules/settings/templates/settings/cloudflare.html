Step 2 — Connect Your Cloudflare Account

Go to the Cloudflare Zero Trust Dashboard, create a tunnel, and copy the install token.

1. Go to Zero Trust → Networks → Tunnels

2. Click "Create a tunnel" → choose Cloudflared

3. Name it (e.g. "home-os")

4. Copy the token from the install command

Step 3 — Configure Public Hostname

In the Cloudflare dashboard, add a public hostname for your tunnel:

1. In the tunnel config, click "Public Hostname"

2. Set subdomain (e.g. home) and select your domain

3. Set Service to: HTTPS://localhost:4443

4. Under "TLS" → enable "No TLS Verify" (self-signed cert)
